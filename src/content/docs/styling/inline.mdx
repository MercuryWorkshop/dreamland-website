You can define inline CSS styles like you would in HTML using the `style` attribute. It accepts an object of CSS properties:

import { Tabs, TabItem } from "@astrojs/starlight/components"

<Tabs syncKey="language">
  <TabItem label="JavaScript" icon="seti:javascript">
```js
const element = html`
  <div
    style=${{
      color: "red",
      backgroundColor: "blue",
    }}
  >
    Hello world!
  </div>
`
```
  </TabItem>

  <TabItem label="JSX" icon="seti:react">
```js
const element = (
  <div
    style={{
      color: "red",
      backgroundColor: "blue",
    }}
  >
    Hello world!
  </div>
)
```
  </TabItem>
</Tabs>
